# 🧠 Gênio Impossible - Edition: Harahel

**Gênio Impossible** é um jogo de quiz de elite projetado para testar os limites do conhecimento humano. Com um visual moderno e mecânicas de gameplay refinadas, este desafio não perdoa erros.

## 🚀 Funcionalidades Premium

- **Visual Cyberpunk:** Interface escura com efeitos neon, vidro embaçado (Glassmorphism) e animações suaves.
- **Mecânica Anti-Memória:** As alternativas são embaralhadas dinamicamente a cada rodada. Onde era "A" agora pode ser "D".
- **Sistema de Ranking:** Hall da Fama persistente que armazena os 5 melhores jogadores localmente.
- **Feedback Tátil/Visual:** Efeitos de *Shake* (tremor) em caso de erro e brilho neon em caso de acerto.
- **Barra de Progresso:** Acompanhamento em tempo real da sua jornada rumo ao título de Gênio.
- **Mobile Friendly:** Hitbox de botões otimizada para jogabilidade em dispositivos móveis.

## 🛠️ Tecnologias Utilizadas

- **HTML5:** Estrutura semântica das telas de jogo.
- **CSS3:** Estilização avançada, Grid Layout, Flexbox e Keyframe Animations.
- **JavaScript (Vanilla):** Lógica do jogo, manipulação de DOM e persistência de dados (LocalStorage).

## 🎮 Como Jogar

1. Insira seu **Nick** na tela inicial.
2. Clique em **JOGAR**.
3. Leia atentamente e selecione a resposta correta.
4. **Cuidado:** Um erro significa fim de jogo e retorno ao menu!
5. Tente alcançar o topo do **RANKING**.

## 🤫 Easter Eggs & Segredos

O jogo contém um código secreto dedicado a grandes criadores de conteúdo. 
*Dica: Se estiver travado em uma questão impossível, tente digitar o nome de um certo fantasma famoso no seu teclado...*

## 📁 Estrutura do Projeto

- `index.html`: O coração do projeto.
- `style.css`: Toda a magia visual e animações.
- `script.js`: O cérebro por trás da lógica e segredos.
- `database.js`: O banco de dados contendo o vasto repertório de perguntas.
